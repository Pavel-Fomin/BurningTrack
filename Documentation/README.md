// TrackList

// TrackList — приложение для управления музыкальной фонотекой на устройствах Apple. Позволяет подключать папки с музыкой, просматривать и воспроизводить треки, управлять файлами (перемещение, переименование), а также работать с метаданными и обложками.

// Основные возможности:
// - Подключение папок с музыкой и построение дерева файлов.
// - Просмотр треков с отображением метаданных (название, исполнитель, длительность).
// - Воспроизведение треков с поддержкой Control Center и Now Playing Info.
// - Операции с файлами: перемещение между папками, переименование.
// - Работа с закладками (bookmarks) для безопасного доступа к файлам и папкам.
// - Кэширование и загрузка метаданных и обложек.

// Архитектура и ключевые компоненты

// - Track — модель трека.
// - PlayerViewModel — управление воспроизведением и взаимодействие с плеером.
// - MusicLibraryManager — управление папками, синхронизация библиотеки, работа с UI-деревом и реестрами.
// - TrackRegistry — хранилище метаданных о папках и треках.
// - LibraryFileManager — операции с файлами треков (перемещение, переименование).
// - View-компоненты: экраны просмотра папок, треков и управления плеером.

// Как запустить проект
// - Требуется Xcode 15.0+ и Swift 5.9+.
// - Склонируйте репозиторий и откройте `.xcodeproj`.
// - Соберите и запустите на устройстве или симуляторе.

// Особенности реализации
// - Для доступа к аудиофайлам используется механизм security-scoped bookmarks.
// - Архитектура построена на паттерне MVVM с четким разделением ответственности.
// - Управление плеером реализовано через собственный PlayerManager с поддержкой Control Center.
// - Работа с файловой системой, метаданными и кэшированием максимально изолирована от UI.
