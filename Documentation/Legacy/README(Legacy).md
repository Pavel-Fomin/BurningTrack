# Legacy Documentation

В этой папке находятся временные документы-доноры предыдущей структуры документации.

Они сохранены только как источник информации до завершения миграции документации.

Каноничные документы находятся в папках:

- `Documentation/Architecture`
- `Documentation/Features`
- `Documentation/Integrations`
- `Documentation/Developer`

После переноса оставшейся полезной информации документы из этой папки могут быть удалены.
